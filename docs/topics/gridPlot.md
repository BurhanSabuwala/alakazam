





**gridPlot** - *Plot multiple ggplot objects*

Description
--------------------

Plots multiple ggplot objects in an equally sized grid.


Usage
--------------------
```
gridPlot(..., ncol = 1)
```

Arguments
-------------------

...
:   ggplot objects to plot.

ncol
:   number of columns in the plot.
